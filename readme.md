Setup:
1. Create Virtual Environment
```python -m venv .venv```
3. Open Virtual Environment
```.\.venv\Scripts\Activate.ps1```
4. Install Requirements
```pip install -r requirements.txt```
5. Run API
[DEV]
```fastapi dev```
[PROD]
```fastapi run```

Format for .env:
```
API_ID="..."
API_HASH="..."
PHONE="+62..."
```
