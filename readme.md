Setup:
1. npm i
2. npm run migration

To Run Project:
`npm start` or just click F5 on VSCode


Prepare .env on root directory:
```
APP_API_PORT=4000
JWT_SECRET_KEY=...

TELEGRAM_API_ID=...
TELEGRAM_API_HASH=...
TELEGRAM_PHONE_NUMBER=+62...

POSTGRESQL_DATABASE_URL=postgresql://[Username]:[Password]@[ServerAddress]/[DatabaseName]

GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
```
