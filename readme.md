Setup:
1. Create Virtual Environment
   "python -m venv .venv"
2. Open Virtual Environment
   ".\.venv\Scripts\Activate.ps1"
3. Install Requirements
   "pip install -r requirements.txt"
4. Run API
   [DEV] "fastapi dev"
   [PROD] "fastapi run"

Format for .env:
```
API_ID="..."
API_HASH="..."
PHONE="+62..."
```
